# Shashi Azad — Software Engineer II Portfolio

Stack: Next.js 14 + Tailwind CSS + Framer Motion + next-themes + next-seo.

## Quickstart
1. Install: `npm install`
2. Dev: `npm run dev` → http://localhost:3000
3. Add `public/resume.pdf`

## Customize
- Edit `data/profile.ts` to change name, bio, skills, projects, contact.
- Colors: `tailwind.config.js`
- Animations: `components/Section.tsx`, `app/globals.css`

## Deploy
- Vercel (recommended): import repo → deploy
- Netlify: build `npm run build`
- GitHub Pages: `npm run export` → deploy `out/`
